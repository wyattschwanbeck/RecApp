# RecApp
A snipping tool inspired screen recorder.
![Demo](demo/demo.gif)
![Latest](demo/demo2.png)

## Features
Croppable video recording <br>
Supportes vertical and horizontal screen orientations<br>
Microphone toggle <br>
Up to 4 displays handled <br>
Minimalistic UI <br>

# RecApp
A snipping tool inspired screen recorder.
![Demo](demo/demo.gif)
![Latest](demo/demo2.png)

## Features
Croppable video recording
Microphone toggle <br>
Up to 4 displays handled <br>
Minimalistic UI <br>
